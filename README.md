# 学習の実行方法

# 各種パラメーターの変更方法

`train.py` にパラメーターをある程度は管理しています．

各種変数の値を変更することによってパラメーターを変更することが出来ます．

モデルの情報(CNNのフィルターの枚数など)は `cnn_model.py`を編集してください.
